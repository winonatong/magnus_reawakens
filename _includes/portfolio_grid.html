About

The Transamerica Pyramid in San Francisco, California was built next to the site of the Ship Niantic. The San Francisco Enlightened believe that this location is of special importance to our understanding of XM and the Shapers. We plan on building a stylized replica of the Transamerica pyramid for Magnus Reawakens in an attempt to communicate with the Shapers.

  • Our Humble Beginnings

    Artist's rendering of the Shaper communication device

  • Blueprints

    The pyramid will be 6 feet tall and constructed of CNC routed plywood. It will be powered by an Android tablet, which provides agents with a touch-screen interface for communicating with the Shapers using glyphs. The pyramid will be illuminated with LEDs, and the tablet will be amplified by a portable bluetooth speaker. This setup can be powered by A/C or by battery if necessary.

    The structure will be adorned with the work of artisans within the Enlightened community, including glasswork and paintings.

  • Proof of Concept

    We built a 1/3 scale model to put our design to the test. The android device runs a fortune telling app that accepts a free form question from an agent using glyphs and returns an answer, also using glyphs. The software on the tablet is based on an open source (GNU GPL) glyphing practice app written by @geckour. The source code for the fortune telling app in progress can be found on GitHub
